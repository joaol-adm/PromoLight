# PromoLight v3.1
- Responsivo: Imagem 1 (60%) + Imagem 2 (40%) na 1ª linha; Imagem 3 (100%) abaixo; vídeo (100%) abaixo.

- Auto-ext: `.svg`, `.png`, `.jpg`.
- PWA com runtime caching e log opcional no GitHub.
